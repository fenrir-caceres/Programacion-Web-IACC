1.- Utilizando un documento HTML, identifica las etiquetas básicas utilizadas para la creación de hiperenlaces y listas. Especifica donde está presente cada una de ellas con sus respectivas descripciones.

Etiquetas de hiperenlaces

Las etiquetas de hiperenlaces son las siguientes:

  • Etiqueta a
        < a href="https://www.google.com/">Etiqueta a < /a>

La etiqueta a es la que se utiliza para crear un hiperenlace, en este caso se utiliza para crear un enlace a google.com


Etiquetas de listas

Las etiquetas de listas son las siguientes:

  • Etiqueta ul
  • Etiqueta ol
  • Etiqueta li
  • Etiqueta dt y dd
        < ul>
            < li>Etiqueta ul< /li>
            < li>Etiqueta ol< /li>
            < li>Etiqueta li< /li>
            < li>Etiqueta dt y dd< /li>
        < /ul>

Lista Ordenada

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
        < ol>
            < li>Elemento 1< /li>
            < li>Elemento 2< /li>
            < li>Elemento 3< /li>
        < /ol>

La etiqueta ol es la que se utiliza para crear una lista ordenada

Lista Desordenada

  • Elemento 1
  • Elemento 2
  • Elemento 3
        < ul>
            < li>Elemento 1< /li>
            < li>Elemento 2< /li>
            < li>Elemento 3< /li>
        < /ul>

La etiqueta ul es la que se utiliza para crear una lista desordenada

Lista Descripción

Elemento 1
Descripción del elemento 1
Elemento 2
Descripción del elemento 2
Elemento 3
Descripción del elemento 3
        < dl>
            < dt>Elemento 1< /dt>
            < dd>Descripción del elemento 1< /dd>
            < dt>Elemento 2< /dt>
            < dd>Descripción del elemento 2< /dd>
            < dt>Elemento 3< /dt>
            < dd>Descripción del elemento 3< /dd>
        < /dl>

La etiqueta dl es la que se utiliza para crear una lista de descripción
